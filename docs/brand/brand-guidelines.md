# Novoriq Brand Guidelines

## Primary Colors

- White: `#FFFFFF`
- Navy Blue (primary brand): `#0B1F3B`
- Slate Grey: `#6B7280`
- Dark Grey: `#111827`
- Light Surface Grey: `#F3F4F6`

## Secondary Colors

- Deep Blue highlight: `#1F3E66`
- Border Grey: `#D1D5DB`

## Typography

Recommended font families:
- `Manrope` (headings)
- `Inter` or `Source Sans 3` (body)

## Logo Usage Rules

- Preferred lockup: icon + wordmark horizontal
- Minimum clear space: at least the width of the checkmark stroke around all sides
- Minimum digital size:
  - icon-only: 24px
  - horizontal: 120px width
- Do not rotate, stretch, or recolor logo outside approved palette

## App Icon Usage

- Use provided `icon.png` for launcher icons
- Use adaptive icon foreground/background files for Android
- Keep icon on clean backgrounds to preserve contrast
